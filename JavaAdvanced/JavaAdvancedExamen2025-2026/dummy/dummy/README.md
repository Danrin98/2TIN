# Dummy Project

Dit project dient enkel om lokaal dezelfde dependencies te hebben als tijdens het examen.
Gezien er op het examen geen internetverbinding is, kunnen de dependencies niet gedownload worden.
Door dit project lokaal te bouwen, worden de dependencies in de lokale Maven repository geplaatst.

Voer het volgende commando uit om de dependencies te downloaden:
```
mvn clean install
```

Na het uitvoeren van dit commando, zijn de benodigde dependencies beschikbaar voor het examen.
Voor alle duidelijkheid: dit project heb je niet nodig tijdens het examen zelf.
